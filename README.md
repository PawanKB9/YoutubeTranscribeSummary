# YouTube Video Summarization App

This is a simple YouTube Video Summarization App built using open-source AI tools.
The app takes a YouTube video, converts the audio into text, and generates a short summary.

It runs on CPU and is made mainly for learning purposes.

## Tools Used

Llama 2 for generating summaries

Whisper for converting speech to text

Haystack for text processing

Streamlit for a simple user interface

## How It Works

User enters a YouTube link

Audio is converted to text

Text is summarized and shown on the screen
